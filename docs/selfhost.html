Quick Start

Parse a document in three commands:

# Install AILANG
curl -fsSL https://ailang.sunholo.com/install.sh | bash

# Clone DocParse
git clone https://github.com/sunholo-data/docparse.git
cd docparse

# Parse a document
./bin/docparse data/test_files/sample.docx

Convert between formats:

# DOCX to HTML
./bin/docparse input.docx --convert output.html

# CSV to DOCX report
./bin/docparse data.csv --convert report.docx

# Markdown to PowerPoint slides
./bin/docparse notes.md --convert slides.pptx

Generate documents with AI:

# Generate a DOCX report from a prompt
ailang run --entry main --caps IO,FS,Env,AI --ai gemini-2.5-flash \
  docparse/main.ail --generate report.docx --prompt "Q1 sales report with revenue table"

Installation

AILANG CLI

DocParse runs on the AILANG runtime. Install it for your platform:

# macOS / Linux
curl -fsSL https://ailang.sunholo.com/install.sh | bash

# Or download directly from GitHub releases
# https://github.com/sunholo-data/ailang/releases

Verify the installation:

ailang --version
# ailang v0.9.2 (or later)

DocParse Source

git clone https://github.com/sunholo-data/docparse.git
cd docparse

# Type-check all 31 modules
./bin/docparse --check

# Run inline tests
./bin/docparse --test

Docker

Run DocParse as an API server with Docker — no AILANG installation needed:

# Pull and run (when published)
docker run -p 8080:8080 sunholo/docparse:latest

# Or build from source
docker build -t docparse .
docker run -p 8080:8080 docparse

The Docker image includes AILANG and all 31 DocParse modules. The API server starts automatically on port 8080.

# Test the running container
curl http://localhost:8080/api/v1/health
# {"result":"{\"status\":\"healthy\",\"version\":\"0.7.0\",...}"}

curl http://localhost:8080/api/v1/formats
# Lists all 13 parse and 8 generate formats
The Docker image builds AILANG from the dev branch for latest features. For production, pin to a specific release tag in the Dockerfile.

Capability Budgets

AILANG enforces hard limits on what each request can do. These are compile-time guarantees — exceed them and the runtime halts deterministically.

CapabilityBudgetPurpose
IO @limit50,000Console output operations
FS @limit5,000File system reads (ZIP entries, output writes)
AI @limit30AI model calls (PDF pages, image descriptions)
NetunlimitedHTTP requests (Firestore, metadata server)
RandunlimitedRandom number generation (API key generation)
ClockunlimitedTimestamp access

AI Models

DocParse uses AI for PDFs, images, audio, and video. Office formats (DOCX, PPTX, XLSX, etc.) are parsed deterministically with zero AI calls.

Google (Vertex AI / AI Studio)

# Via Application Default Credentials (Cloud Run, GKE)
GOOGLE_API_KEY="" ailang run --entry main --caps IO,FS,Env,AI \
  --ai gemini-2.5-flash docparse/main.ail document.pdf

# Via API key (local development)
GOOGLE_API_KEY="your-key" ailang run --entry main --caps IO,FS,Env,AI \
  --ai gemini-2.5-flash docparse/main.ail document.pdf

Anthropic (Claude)

ANTHROPIC_API_KEY="sk-ant-..." ailang run --entry main --caps IO,FS,Env,AI \
  --ai claude-haiku-4-5 docparse/main.ail document.pdf

Ollama (Local, Free)

# Start Ollama and pull a model
ollama pull granite3.2-vision

# Run DocParse with local model (no API key needed)
ailang run --entry main --caps IO,FS,Env,AI \
  --ai granite-docling docparse/main.ail document.pdf
Recommended model: gemini-2.5-flash — best balance of accuracy (92% on PDF benchmark) and speed. Ollama models score lower (0-3%) due to structured JSON output limitations.

Benchmarks

DocParse maintains 53 golden benchmark files across all 10 deterministic formats with 100% structural recall.

# Quick smoke test (~15 seconds)
bash benchmarks/quick_check.sh

# Full 53-file structural benchmark (~10 minutes)
uv run benchmarks/run_benchmarks.py --suite office

# API server tests (32 tests)
bash tests/test_serve_api.sh

Key performance numbers (Apple M2):

  • Sequential DOCX parse: ~11ms per file
  • Concurrent 5x DOCX: ~26ms total (near-perfect scaling)
  • 53-file benchmark: 100% structural recall across all formats
  • PDF (Gemini 2.5 Flash): 92% accuracy on OmniDocBench subset
