Office Structural Benchmark (OfficeDocBench)

Most benchmarks focus on PDF extraction accuracy. Nobody benchmarks Office parsing quality — whether a parser preserves track changes, merged cells, comments, headers/footers, and text boxes.

OfficeDocBench is the first benchmark focused on structural Office parsing quality. 59 test files across 10 formats, scored on 5 metrics: feature detection, structural recall, text Jaccard, element count accuracy, and metadata extraction. Run date: 2026-03-20.

Run the Benchmarks

# OfficeDocBench — formal benchmark with scoring (instant, uses golden files)
uv run benchmarks/officedocbench/eval_officedocbench.py

# OfficeDocBench — with competitor comparison
uv run benchmarks/officedocbench/eval_officedocbench.py --all

# Quick regression test (~15 seconds)
bash benchmarks/quick_check.sh

# Full Office structural regression (~10 minutes)
uv run benchmarks/run_benchmarks.py --suite office

# PDF benchmark (needs AI model)
uv run benchmarks/run_benchmarks.py --suite pdf --ai gemini-2.5-flash

# Regenerate golden outputs after parser changes
bash benchmarks/generate_golden.sh
