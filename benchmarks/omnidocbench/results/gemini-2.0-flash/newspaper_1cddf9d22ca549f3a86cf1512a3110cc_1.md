# 要闻·综合

本报讯（记者 李增民）近年来，山东省农业信息化建设取得长足发展，建成覆盖省、市、县三级的农业信息网络体系，初步实现了农业信息管理的规范化、智能化和科学化，为转变农业发展方式，推进农业现代化建设奠定了良好的基础。

省农业厅有关负责人介绍说，山东省农业信息资源库已基本形成，可提供政策法规、市场行情、科技知识、病虫测报等各类信息；完善了全省农业数据采集、传输、存储和处理等软硬件系统，初步实现了农业数据信息的“集中管理”，在抗灾救灾、生产调度、市场监管等方面发挥了重要作用；搭建了集农业新技术集成示范、农业科技成果转化、农业信息培训和专家在线指导为一体的“山东三农信息服务平台”，为基层农技推广人员和农民群众提供了方便快捷的信息服务；建设了省、市、县三级农业部门网站和农产品市场信息直报点，建立农业数据共享制度，健全农业信息发布评估体系，规范农业信息发布内容和形式，努力提高农业信息发布能力。

农业信息化建设的推进，提高了全省农业管理服务水平。目前，该省农业部门已基本完成行政审批服务事项的网上运行，形成了覆盖省、市、县、乡、村五级农业信息化体系，农业农村信息化成为各级农业部门提高行政管理和决策水平、服务“三农”的重要手段。

据了解，山东省农业信息办与省气象局合作建立了农业气象服务平台，为农业抗灾救灾提供了有力支持；与省统计局合作建立了农产品产量预测预警系统，提高了农产品产量预测的准确度。此外，山东省还积极推动物联网、云计算等现代信息技术在农业领域的转化和应用，为加快农业信息化建设进程奠定了基础。

# 财政助农 福建茶农迎来又一春

■ 本报记者 蔡茂祥

福建拥有“茶之乡”、“茶之都”等诸多美誉。2008年以来，中央财政投入巨资，福建省委、省政府积极配合，中央、地方财政共计投入资金达2.29亿元，整合相关部门资金4.33亿元投入茶产业，中央和地方财政的扶持，有效拉动了福建现代茶产业的发展。

“经过一年，可以说是一年一个新台阶，这个新台阶发展迅速，主要得益于中央财政资金的扶持。”福建省茶叶学会会长冯廷佺介绍说，福建省的茶叶是传统优势产业，又是全国重要产茶省份，全省茶叶种植面积达300万亩，居全国第四位，茶农年终总收入超过300亿元，茶叶产业已成为福建省农民增收致富的支柱产业之一，在全国占据重要位置。

中央财政资金的政策性作用，促进了福建省生态茶园和茶叶品种的优化提升，促进了质量安全体系的建设，提高了茶叶的经济效益和竞争力。改善生态，构建现代茶产业发展基础。

一条条宽敞的机耕路四通八达，装有程控通讯设备、安装了“小喇叭”的茶园，依偎在水库、港口附近的蓄水池、星罗棋布……这是记者在松溪县溪东现代茶业园见到的景象。自从2008年省里将松溪县列入现代茶业生产建设项目县后，全县茶叶生产基础设施就逐渐完善。规模以上2000亩以上的茶叶生产园区就达到10家。

茶园的生态环境直接影响着园区茶叶产业的发展。据介绍，福建省投入资金标准化生态茶园建设，制定了详细的补助标准。2008年——2010年，全省共建设标准化生态茶园达29.35万亩，为福建茶叶的可持续健康发展奠定了良好发展基础。

据介绍，福建茶叶产区大多分布在海拔较高的山区。早些年，福建有关茶叶品种的优异性尚未得到充分挖掘，茶叶品种资源比较单一，不利于当地茶农增收。自2008年以来，福建省茶叶产业得到了充足的资金，鼓励各地大力发展名优茶新品种。目前，福建省茶叶学会已经收集储备了大量的茶叶品种，为福建名优茶的普及奠定了坚实基础。

据介绍，福建省茶叶产业的优势在于名茶，比如铁观音等茶叶在全国都享有盛誉。但在品种的优化方面，福建省仍然需要改进。2008年——2010年，全省共引进、选育茶树良种达95个，省级良种21个，无性系良种茶园面积达9.13万亩，高于全国35%的平均水平。居全国首位。全省平均茶叶单产92公斤/亩，高出全国85.7公斤/亩，居全国第一。全省名优茶产量9.53万吨，产量75亿元，分别占全省茶叶总产量和总产值的33.3%。

提升质量，奠定茶叶的增收基础

记者来到全国顶级大乌龙茶生产区安溪县，在龙娟合作社的生产车间，记者看到，茶农们正忙着采摘茶叶，车间里弥漫着浓浓的茶香。合作社负责人介绍说，为了保证大乌龙茶的品质，合作社组织茶农统一标准、分级包装，杜绝了不正当竞争，保障了茶农的利益，同时也提升了茶叶的质量。

自现代茶业项目在安溪实施以来，该县通过建立测报预警点，配备TEC型太阳能杀虫器、诱虫灯，随时监测虫情，生物农药被积极推广，机动喷雾器等农业机械也被广泛应用。

不仅在安溪县，几乎所有重要的中央财政支持的茶叶主产区，都提升质量安全意识。为了茶叶生产的安全，为每株茶树开展追溯，建立了农药使用、项目实施区域建立无公害茶叶病虫害测报点，促进制定完善的测报体系，指导茶农科学防治病虫害，构建了道道茶叶质量安全防线。

茶叶质量安全的提升，促进了茶农总体的增收。2009年，福建全省农民人均纯收入为6800.2元。比2007年增加1213.1元，再增长22.2%，安溪、福安、武夷山等茶叶主产区农民人均纯收入人口达60%-70%，农民人均纯收入中的50%-60%来自茶产业。

整合资金，带动优势特色茶产业发展

“现代茶业资金周转加快，增加了福建茶叶的附加值，解决了福建茶业产业结构单一，促进茶农增收困难等问题。”福建省农业厅的相关负责人介绍说，福建是国家海峡西岸经济区，面临着良好的发展机遇。但由于没有相关政策的积极扶持，福建茶叶难以得到持续发展。通过积极协调，省政府协调资金，成立茶叶产业协会，合作社组成行业协会，对每个组织给予政策性支持，鼓励企业互相沟通交流，建立茶叶产品集散中心，整合资金，提高福建茶叶的规模化，促进优势特色茶叶产业发展。

2009年，福建省茶产业又以快速发展的态势向前发展，全省茶叶种植面积达到300多万亩，年总产量达到20万吨。在茶叶产业的带动下，福建省农民人均纯收入已接近7000元，县以上龙头企业达到300多个，省级以上的龙头企业10多个，省级以上龙头企业资产高达300亿元，年销售额达90亿元，上缴利税达1亿元。

据介绍，今后，福建省计划继续加大茶叶产业的投入力度，一方面，继续对优势茶产区的茶叶加工企业、专业合作社进行扶持，另外，对优质茶叶的生产加大支持力度。有关负责人表示，为了巩固已有的成绩，确保农民增收，茶农致富，需要加大力度，推动农业产业的综合发展，加大力度，提高福建茶产业的品质。

中略微 蔡茂祥
